# BA Agent - Prompt Templates

## User Story Template

### Create User Story
```
As a [user type],
I want to [action/feature],
So that [benefit/value].

**Acceptance Criteria:**
- [ ] Given [precondition], when [action], then [result]
- [ ] Given [precondition], when [action], then [result]

**Definition of Done:**
- [ ] Feature implemented
- [ ] Unit tests passing
- [ ] UI matches design
- [ ] Accessibility verified
- [ ] Documentation updated
```
